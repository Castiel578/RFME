# Recursive-Frechet-Mean-Estimator

This repository contains the source code and implementation for the paper **"Recursive Fr\'echet Mean Estimator"**. The provided codes reproduce the simulation experiments and real-world data analyses presented in the paper.

The repository is organized as follows:

- **`simulation/`**: Contains files for the simulation studies on different manifolds, including experiments on the sphere \(S^2\) and the special orthogonal group \(SO(n)\).

- **`real_cases_study/`**: Contains the code for empirical evaluations on real datasets, including the MPEG-7 dataset and the ANDI database.

These experiments compare the performance of the empirical Fréchet mean estimator, the incremental Fréchet mean estimator, and the proposed recursive mean estimator.
